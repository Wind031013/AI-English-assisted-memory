使用vue3框架结合element_plus框架写一个界面，以及使用fastapi实现后端接口
我的需求做一个英语辅助记忆平台详细需求如下：
第一个界面需求：
单词书的数据格式如下：
{
    "id": 1,
    "name": "四级英语单词",
    "description": "四级相关单词书....",
    "words": [
        word1,
        word2
        ...
    ]
}
是单词书选择界面，展示所有的单词书如：《高考英语单词》，《四级英语单词》....等以及可以自己上传单词书并进行命名，这些数据都存放在数据库中通过fastapi提供
第二个界面是辅助记忆页面
单词书列表中每个单词的格式如下
{
    "word": "study",
    "translation": {
        "v.": "学习",
        "n.": "研究"
    },
    "example": {
        "text": "I am studying.",
        "translation": "我正在学习。"
    },
    "remember": false
}
功能需求如下：
有三个不同列表组成分别为单词列表,记忆列表,最终列表三个列表的具体功能如下
- 展示单词列表中的单词时,仅展示此单词以及"认识","不认识"两个按钮
    - 点击认识后
        - 翻转卡片显示单词的翻译,并且显示"下一个"，"记错了"两个按钮
            - 当点击"下一个"后,单词进入记忆列表
            - 当点击"记错了"后,切换下一个单词
    - 点击不认识后
        - 翻转卡片显示单词的翻译,显示"下一个"一个按钮
- 展示记忆列表的单词时,展示此单词以及四个翻译(一个此单词的翻译，以及3个类似单词的翻译)的选项
    - 选择正确后
        - 此单词进入最终列表，直接显示下一个单词
    - 选择错误后
        - 翻转卡片显示单词的翻译,并且此单词进入单词列表,显示"下一个"按钮
- 展示最终列表的单词时,最终列表会展示那个单词的例句,可点击按钮"认识"、"不认识"两个按钮"
    - 点击认识后
        - 翻转卡片显示单词的翻译以及例句的翻译,并显示"下一个"，"记错了"两个按钮"
            - 当点击"下一个"后,从最终列表移除,并给后端发送通知标记这个单词的remember为true
    - 点击不认识后
        - 翻转卡片显示单词的翻译，此单词重新进入记忆列表,并显示"下一个"一个按钮，
选择单词书进入辅助记忆页面后后端随机从单词书中选择10个未记单词放入单词列表中，并随机展示单词列表,记忆列表,最终列表其中之一中的单词
将两个界面分别写在不同的两个组件中使用router-view进行切换
